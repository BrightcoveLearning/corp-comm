Controlling Access to an Experience

In this topic you will learn how to restrict access to an experience using access control profiles.

Access Control Profiles are used to control access to experiences and are created as part of the Project settings. Once profiles have been created, they can be assigned to experiences to control access. For information on creating Access Control Profiles, see Creating Access Control Profiles.

To assign an Access Control Profile to an experience, follow these steps:

  1. Edit the experience.
  2. Click ACCESS CONTROL in the left navigation.
  3. Select an access control profile for the experience.
  4. Click Save.

Make sure you republish the experience after making any changes to the access control settings.
