Creating and Managing Push Notifications

In this topic you will learn how to create and manage push notifications.

A push notifications is a message that pops up on a mobile device. Push notifications are a convenient way to notify users about new content or upcoming live events that are/will be available in the Brightcove Engage™ app. When a mobile user clicks a push notification, they will be directed to the new content inside the Brightcove Engage app.

Note: Push notifications can only be created as part of a Mobile App experience.

Creating notifications

To create a new push notification, follow these steps:

  1. Edit the Mobile App experience.
  2. Click NOTIFICATIONS in the left navigation. A list of notifications that have been sent or are scheduled will display.
  3. Click Create Notification.
  4. Enter a Notification Title and Description. These will be displayed on the device so having concise, detailed description is recommended.
  5. The On click send to: section determines what happens when the notification is clicked on the device:
    • Application Home - The user is directed to the home page of the app
    • Collection - The user to the selected collection
    • Video Content - The user is directed to the selected video; selecting the Include video thumbnail? option will include an image with the notification

The Preview area will display a preview of what the notification will look like when sent.

Click Send to send the notification immediately or Schedule, select a date and time and then click Confirm.
